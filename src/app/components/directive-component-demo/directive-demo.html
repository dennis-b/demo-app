Angular Directive simple example.

This is regular angular directive that using: @Directive annotation
You can specify link/compile/controller functions
Create View directive. by default using Class name as Directive name : 'ng-directive-demo'
Or you can specify selector in View options
import './style/directive-demo.css' import template from './directive-demo.html'; import {Directive} from 'annotations/directive-decorator'; @Directive() class NgDirectiveDemo { constructor($window) { this.restrict = 'EA'; this.template = template; this.scope = { items: '=' }; this.$window = $window; // service injected } link(scope, element, attrs){ } } export default NgDirectiveDemo
